ITIS 3310 - Software Architecture & Design

Filters

Description:

This course introduces the fundamentals of a discipline of software engineering known as Software Architecture & Design. This involves an inter-disciplinary approach to building large-scale enterprise software systems with an emphasis on both process and decision-making in the construction of high-quality software. This will include topics on software design with emphasis on architectural design and design patterns. Also included: architecture styles and patterns, as well as use of non-functional requirements for tradeoff analysis. A system design language, known as UML (or Unified Modeling Language), is introduced and used throughout the course. Because software engineering is in a state of constant change and disruption, an emphasis on integrating new concepts such as agile, cloud, and devops will be introduced to help modernize the foundational elements of this course as they apply to software architecture & design.

Prerequisites ITSC 2214 - Data Structures and Algorithms
Language/s None
IDE/API Platform None
Topics Describe how not just to build software systems, but build QUALITY software systems using sound architectural & design principles and practice • Describe & model software using UML with Visio or Draw.io* • Demonstrate foundational understanding of Agile Software Development Methodologies • Apply a confident, contextual use & application of terms like: abstraction, traceability, & patterns • Think critically about architecture & design concepts regardless of your IT or business role • Integrate & adapt to future changes in software development methodologies and technologies
