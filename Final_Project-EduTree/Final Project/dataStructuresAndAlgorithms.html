Data Structures and Algorithms

Filters

Description:

A study of the theory and implementation of abstract data types (ADTs) including stacks, queues, and both general purpose and specialized trees and graphs. Includes the implementation and analysis of algorithms related to the various data structures studied, including creation, searching, and traversal of ADTs.

Prerequisites ITSC 1213 -- Introduction to Computer Science II
Language/s Java
IDE/API Platform NetBeans
Topics Algorithms, Array, ArrayList, Best Case/Worst Case, Big O notation, Binary Search Trees, Data Structures, Graphs, Heaps, Linked List, Nodes, Queues, Run-time complexity, Recursive Algorithms, Searching, Stacks, Traversal, Trees
